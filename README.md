Django-Test
===================
**Mi primera implementación de Django**

### Instalación ###

* Clonar repo
* `git clone https://github.com/drcrow/Django-Test.git projectname`
* `cd projectname`
* Migrations
* `python manage.py makemigrations productos`
* `python manage.py migrate`
* Iniciar server
* `python manage.py runserver`
